# BRIGHT-TV-CASE-STUDY
• This is a project focusing on gaining insights on user and usage trends of BrightTV

• What type of factors influence consumption?

• What content would I as the analyst recommend to increase consumption on the days with low.
consumption?

• What type of initiatives would I recommend to further grow BrightTV ‘s User base.

####
THE OBJECTIVE
BrightTV ‘s CEO has an objective to grow the company’s subscription base for this financial
year. He has approached myself to provide insights that would assist CVM (Customer Value
Management) team in meeting this year’s objective.

###
INTRODUCTION
Bright TV has been having been struggling with viewership base increasing and based on the data provided the CEO 
has requested we extract insights on ways Bright TV can increase numbers of customer base and improve content to retain exsisting viewrs too.
###

HOW I DID THE CASE STUDY AND WHAT INSIGHTS DID I EXTRACT
• Project planning was done using MIRO for brainstorming ideas on converting the raw data into valuebale insight.

• Data Ingestion was done on Data Bricks and the Data Management System along with the SQL code script.

• Data Visualisation along with extensive graphs drawn using Pivot table on Excel.

• Project time-lines was done using CANVA in order to set the tone on how the project will be processed.

### Key insights retrived include:
• Data set showed AGE DIFFERENCES Youngest is 0 AND Oldest 114 years old.

• Province 9 PROVINCES and 1 NONE Gauteng has the highest number of Viewership 3654 and Northern Cape the lowest with 230.

• There are 10000 viewers on Data set and 8761 are male 977 are Female with 262 None.

• The ICC Cricket World Cup 2011 was the most watch channel over the 3 month period.

• Wednesday's and Fridays have the highest viewership by day name over the 3 month.

### Recommendations
• After careful data extraction and review I advise we focus on revamping content on low performing channels, by conducting questionnaire to all subscribers with a particular focus on gathering information on viewership trends in the Northern Cape.

• Bright TV has information such as social media handles and emails, along with Broadcast slots that can be used to advertise popular channel events for days with slow viewership to attract clients and spread the word for new subscription.

• Investigate the Cost Benefit Analysis for the(Customer Value Management) CVM Team, to creating a Bright TV APP that will reach Viewership in an unconventional way to improve channels viewership base.
